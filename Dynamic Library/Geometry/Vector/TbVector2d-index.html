class TbVector2d

Клас призначений для зберігання значень вектора на площині (його координат).

Поля
Доступ Тип Назва Опис
private double x x - координата на площині.
y y - координата на площині.

getRadianBetween(const TbVector2d *v)const Повертає кут в радіанах між променем, що утворює вектор та променем, що утворює вектор v у межах [0; pi].
